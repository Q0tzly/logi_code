# Basic Expression
All expression is write with polish notation.
Unuse brackets.

## Operator
Support only two operators.

### OR
`or A B`
- Use two arguments.
- Return OR with A and B.

### NOT
`not A`
- Use one argument.
- Return not of A.

## Literal
`A : 0`
- Use `0` and `1` as literal of bool type.
